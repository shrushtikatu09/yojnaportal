# yojnaportal
Yojana Portal is a simple and user-friendly web application that provides access to over 20+ government schemes. Users can sign up, log in, and enter their personal details—such as age, region, and other information. Based on their eligibility, the portal suggests relevant government schemes for which they can apply. The portal is built using [Full-Stack Web Application | MongoDB, Express.js, React.js, Node.js (MERN)] and offers a seamless experience by connecting users to official government websites for each scheme, ensuring accurate and up-to-date information.
